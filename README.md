# DApp Starter

## How to clone

```sh
git clone --recurse-submodules git@github.com:jakim929/dapp-starter.git
```

## How to run locally

### install dependencies
```sh
pnpm i
```

### run dev environment
```sh
pnpm dev
```
the above command will
1. start anvil instance
2. deploy contracts (foundry)
3. start dev server for frontend (vite)
